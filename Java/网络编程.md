## 在java中发送UDP数据包

### 类# InetAddress

这个类封装了IP地址

![[Pasted image 20250507103925.png]]

用这个可以封装一个IP地址给其他相关函数使用


![[Pasted image 20250507102246.png]]


DatagramSocket这类是用来发送数据包和接受数据包的套接字

数据报套接字是数据包传送服务的发送点或接收点。在数据报套接字上发送或接收的每个数据包都被单独寻址和路由。从一台机器发送到另一台机器的多个数据包可能会有不同的路由，并且可能以任何顺序到达。

一个套接字由IP和端口号组成，可以用来唯一确定进程之间网络通讯的地址

DatagramPacket这个类则是用来封装数据包的

这种方式仅能传送UDP数据包
