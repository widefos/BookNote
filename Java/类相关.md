## 内部类

一个类的内部可以定义另一个类，内部类可以访问外部类的属性和成员方法

但是**外部类无法直接访问内部类的成员**

![[Pasted image 20250427173401.png]]

内部类是依托于外部类而存在的，所以说内部类是外部类的一部分

## 访问修饰符

![[Pasted image 20250511134315.png]]

private只允许在当前类内访问

protected允许在类内和继承的子类内访问

public则允许所有访问
