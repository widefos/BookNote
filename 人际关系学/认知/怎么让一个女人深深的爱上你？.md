## 1. 什么是深爱？警惕病态深爱

普遍认为，有钱、帅气、身材好就会有女孩喜欢。但这只是"喜欢"，不是"深爱"。

**什么是深爱（病态版）？**

| 状态 | 表现 |
|------|------|
| 他做的对的 | 那是优秀 |
| 他做的错的 | 也是对的 |
| 他错得非常离谱 | 肯定有他的难处 |
| 他打我 | 是我做得不好，惹他生气了 |
| 他劈腿了 | 是我对他不好，我要改 |
| 他要跟我分手 | 是他太优秀了，我配不上，我努力改 |

> 妹子，你有没有过这种感受？如果有，你的确是深爱他，而且是**极端的病态深爱**——你中套路了。

## 3. 手段一：服从性测试

> 说是服从性测试，不如说是**让你养成服从的习惯**。

这个习惯从刚刚恋爱开始，逐渐加深、加重，日常进行，潜移默化。

| 层级 | 难度 | 示例 |

| **初级** | 随手能做的事或随口说的话 | 喂你东西时到嘴边说："叫哥哥（老公），叫就给你吃" |

就像诈骗犯从初级测试开始，逐渐加深，最后让女性深信不疑。

## 4. 手段二：推拉

> 就是对你时好时坏，热三天，冷三天。

**目的：** 让你的情绪上下起伏。

- 谁能让女人情绪起伏，女人就会依赖谁
- 起伏得越厉害、越频繁，依赖得就越严重

### 关键区分

| 手段 | 效果 |
|------|------|
| **正确推拉：起 + 伏交替** | 情绪高涨 → 低落 → 高涨交替，最让人欲罢不能 |
| **纯打击（只有伏）** | 情绪始终趴着，时间久了你会厌倦 |

### 危险信号

他非常擅长做这种事情：

- 每当你情绪低落到快要崩溃 → 他回来拉你一把
- 每当你非常兴奋 → 他又给你整幺蛾子

> **警示：** 像这种频繁玩推拉的，别管他是不是故意的，**当机立断，马上分手**，是你的最佳选择。

## 6. 复合操作：温水煮青蛙三阶段

**甜言蜜语的难点：** 短期内快速了解对方，说出对方最爱听的话，一般人很难做到。

2. 他让你服从爱的命令（初级服从性测试）
3. 你服从了 → 他更卖力地赞美你、夸你
4. 你更开心 → 快速进入热恋期

```
他开心，又开始对你好、捧你 → 你开心
     ↓
（循环）
```

> **潜意识变化：** 你的快乐已经慢慢地和"听从命令"挂钩了。未来你会越来越听话，对他的依赖性越来越强。
