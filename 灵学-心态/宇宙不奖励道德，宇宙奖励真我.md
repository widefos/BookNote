> **宇宙不奖励道德，宇宙奖励真我。**

## 核心观点

世界并不会奖励那些善良、努力、勤奋的人，世界只会奖励一种人——**尊重并顺应自己感受的人**。

看似是一个心理学问题，但实际上心理学的尽头是神秘学。

---

## 道德 vs 真我

| 道德（社会标准）                      | 真我（宇宙标准）       |
| ----------------------------- | -------------- |
| 人类社会创造并约束行为                   | 宇宙没有道德，只有能量高低  |
| 不同时代道德感不同（克己复礼 → 劳动光荣 → 有钱至上） | 能量高低取决于你是否心口如一 |
| 为了符合他人期待而行动                   | 为了内心的真实感受而行动   |
| 耗能、低频、不快乐                     | 高频、平静、有热情      |

### 宇宙的规则

- 宇宙**没有道德高低**，只有**能量高低**
- 如果你发自内心想做一件事，做这件事让你专注、兴奋、孜孜不倦 → **能量状态高** → 宇宙会奖励你
- 它会吸引更多你专注的事情和帮助你能的人来到身边

---

## 两种赚钱心态的对比

| 发自内心想赚钱    | 为了符合道德而赚钱    |
| ---------- | ------------ |
| 对赚钱有浓厚的兴趣  | 只是为了养家糊口     |
| 做这件事专注、兴奋  | 为了不被人看不起     |
| 能量高 → 宇宙奖励 | 为了符合社会标准约束自己 |
| 吸引资源和机会    | 牺牲自己，成全社会期待  |

> 如果连你自己都牺牲自己，不在意自己真实的感受，那么宇宙将不会奖励你。

---

## 案例：张雪

> 玩摩托车并不赚钱，也不符合社会世俗成功的标准，甚至对于一个穷人来说并不现实，但是张雪足够听从内心的声音。

- 做这件事他是**开心、期待、有激情有梦想**的
- 就算不成功，他也能成为**一个开开心心的小老头**
- 他不知疲惫，能量高涨 → 身边的人都能感受到这股能量
- 源源不断的人借钱给他、帮助他

> **你足够想做一件事的时候，全世界的人都会给你让步，因为你的能量太强了。**

---

## 如何保持好磁场

核心：**听听你内心的真实感受**

| 真实的状态      | 违心的状态         |
| ---------- | ------------- |
| 做这件事开心、平静  | 做这件事只是为了回报或索取 |
| 善良是纯粹不求回报的 | 善良只是出于道德约束    |
| 种下真正的善因    | 身体不舒服，在耗能     |

---

## 总结

> 祝大家可以成为一个高能量的人，成为一个真我的人。
