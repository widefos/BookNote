## 基础命令


### mkdir 文件夹创建

```
mkdir 文件名 //在当前目录下创建文件夹

mkdir 路径+文件名  //在指定路径下创建文件夹

mkdir /路径/文件1 文件2 //隔着一个空格可以一次创建两个文件

```


### touch 创建文件
```
touch 文件名 //创建一个文件，文件需要加上后缀，例如a.txt表示创建的是txt文件
```



### rm删除

```
rm a.txt //删除一个文件
rm abc -r //删除文件夹
rm * -r   //删除所在目录下的所有文件（不包括隐藏文件）
```

### cp拷贝

```
cp 1.txt 2.txt 
cp abc abc_bak -r 拷贝文件夹
```

### mv移动

```
mv 1.txt a.txt //将第一个文件的名字重名为为第二个
mv 1.txt 目录 //将文件移动到指定目录下
```
