双指针的概念，在链表或者数组中使用两个指针变量来完成相关操作，可以降低时间复杂度，优化算法



**题目**：原地移除数组中所有等于 $val$ 的元素,要求不能使用额外的辅助空间,即空间复杂度为$O(1)$,
返回移除元素后新数组的$size$。

暴力解法：
思路：用一个for循环遍历整个数组，另一个for循环将后面的数组元素向前移动

```
class Solution {
public:
int removeElement (vector<int>& nums, int val). {.
int size = nums.size();
for (int i = 0; i < size; i++) {
if (nums[i] == val){//发现需要移除的元素就将数组集体向前移动一位”
for (int j = i + 1; j < size; j++) {
nums [j - 1] = nums [j];

i --; //因为下标i以后的数值都向前移动了一位,所以i也向前移动一位
size --; //此时数组的长度-1

return size;
```
时间复杂度为$O(n^2)$


双指针解法:
```
int removeElement (vector<int>& nums, int val). {
	int slowIndex = 0;
	for(int fastIndex = 0;fastIndex<nums.size();fastIndex++){
		if(val != nums[fastIndex])
			nums[slowIndex++] = nums[fastIndex];    //这里的逻辑是当在前的指针和目标值不相同时就将元素移动到上一个和该元素相同的位置，这样和val值相同的位置就被后续元素覆盖了
	}
	return slowIndex;
}
```
时间复杂度为$O(N)$

**题目：** 在一个正整数数组nums中找到最小长度的连续子数组,使子数组元素之和大于或等于s,
返回满足条件的连续子数组的最小长度,如果没找到则返回0。


暴力解法:


![[Pasted image 20250325172822.png]]
对于暴力解法来说，依然是用两个for循环来完成，第一个循环更新left，第二个循环用来找到此时
sum>target的最小区间长度，完成后将left向前移动，right归位继续

这样的时间复杂度就是$O(N^2)$

**怎样优化?**
造成时间复杂度提高的关键点在于**每一次更新都需要将right归位**，而此时显然sum>=target,后续right仍然需要经过这些位置，所以我们让right不回头继续向前走而在每一次循环中调整left的值，因为我们只需要找到最小值，所以我们只需要移动left到让sum>=target成立的最小区间长度即可

这样最多操作次数就是 $2N$ 次，时间复杂度被降到了$O(N)$

双指针移动的具体逻辑：

1.  将right向前移动
2. 判断区间内的值是否为sum>=target，如果是则将left向右移动，缩小区间的值，并更新区间最小值
这样最后得到的就是区间最小值

双指针写法：

```
class Solution {
public:
	int minSubArrayLen (int s, vector<int>& nums) {
	int result = INT32_MAX;
	int sum = 0;
	int i = 0;
	int subLength = 0;
	for (int j = 0; j < nums.size () ; j++) {
		sum += nums [j];
		//注意这里使用while,每次更新i(起始位置)并不断比较子数组是否符合条件
		while (sum >= s) {
			subLength(ji+1);// 获取子数组的长度
			result = result < subLength ? result : subLength; //更新最大值
		// 这里体现出滑动窗口的精髓之处,不断变更1(子数组的起始位置)
			sum +=nums [i++];
			}
		}
	return result;
}
```

这题的逻辑和上题类似，采用双指针的方法
指针移动逻辑：
1. right向右移动，并更新每个字符出现的次数
2. 当right所指的字符在当前窗口内有重复出现时，将left向右移动，更新窗口内该字符的出现次数（次数-1）
3. 更新不重复子串的最大值(**注意这里更新最大值和上一题不同要在外循环中更新)**

代码：
```
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        int a[128]={0};//数组模拟哈希表判断字符是否只出现一次
        int n=s.size(),ret=0;
        for(int left=0,right=0;right<n;right++)
        {
            a[s[right]]++;//标记字符出现的次数
            while(a[s[right]]>1)//字符如果出现两次就出窗口
            {
                a[s[left]]--;
                left++;
            }
            ret=max(ret,right-left+1);//更新最长区间长度
        }
        return ret;
        
    }
};
```

**总结：对于双指针来说，关键在于确定两个指针如何移动以及如何对区间当中的相关量进行计算，对于数组操作来说，如果时间复杂度为$O(N^2)$,那么就可以考虑双指针来优化**.
