题目描述：在一个正整数数组nums中找到最小长度的连续子数组,使子数组元素之和大于或等于s。返回满足条件的连续子数组的最小长度,如果没找到则**返回0**。

二分查找代码：
写法一：开区间的搜索方法
```
int search(vector<int>& nums, int target) {

        int left = 0;

        int right = nums.size()-1;

        while(left<=right){   //对于边界情况的处理，这里搜索的区间范围为闭区间[]，所以当left==right的时候是有意义的

             int mid = left+(right-left)/2; //这里是为了不溢出等同于相加除以2

             if(nums[mid]>target)

             right = mid-1;

             else if(nums[mid]<target)

             left = mid+1;

             else if(nums[mid] == target)

             return mid;

        }

        return -1;
```

需要注意的是二分查找针对的是有序数组序列，时间复杂度为$O(log(N))$,$N$为数组元素个数
