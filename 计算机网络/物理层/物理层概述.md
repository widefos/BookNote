# 数据通讯基础

### 相关术语

**数据（data）**：运送消息的实体
**信号（signal）**：数据的电气或者电磁表现，比如模拟信号和数字信号
**码元**：使用时间域的波形表示数字信号时，则代表不同离散数值的基本波形就是码元，通俗解释就是用一段时间内的波形表示特定的数据

**信道**：向一个方向传送信息的媒介，平常我们所说的通信线路往往包含一条发送信息的信道和一条接受信息的信道
**单向通信（单工通信）**：只能有一个方向的通信儿反方向没有交互
**双向交替通信（半双工通信）**：两个方向都可以通讯可以交互，但是双方不能同时发送信息，比如打电话不能同时说话
**双向同时通信（全双工通信）**：通讯双方可以同时发送和接受信息

**基带信号**：来自信源的信号，比如计算机直接输出的各种信号
**带通信号**：将基带信号经过载波调制后，提高信号的频率从便于远距离传输

所以在传输过程中如果两台计算机相隔较近，那么一般用基带传输，远距离一般采用带通信号传输

### **编码方式**：
![[Pasted image 20250327154338.png]]

用不同的电平信号来表示比特流

1. 不归零制：高电平代表1，低电平代表0，这样表示虽然简洁但是却无法区分相连的0和1
 ![[Pasted image 20250327155033.png]]
2. 曼彻斯特编码：由于不归零制存在无法识别连续的0和1的问题所以有曼彻斯特编码，用电平跳变的方式表示0和1
![[Pasted image 20250327155213.png]]
3. 差分曼彻斯特编码
![[Pasted image 20250327155558.png]]
差分曼彻斯特要注意区别，如果中间有信号跳变并且两个时间间隔之间也有信号跳变就是0，如果两个时间间隔没有信号跳变则表示1，**确定当前时间间隔是0还是1要看上一个时间间隔有没有跳变和时间间隔之间有没有跳变**

# 信道复用技术

**复用的概念**：当发送端和接收端有多个如果他们都使用同一条信道那么就是信道的复用

### 频分复用技术

当有多路信号需要传输时可以将这多路信号调制为不同频率的，从而使得**彼此互不干扰，所有信号可以同时发送和接受**

典型例子就是收音机对讲机接收和发送信号，有很多不同频率的信号共用同一个信道，接受方只要选择接收特定频率的信号即可


### 时分复用技术

时分复用则是将时间划分为一段段相同的时间帧，然后给每个用户分配给一段时间帧，每个用户占用的时间片段周期性的出现，时分复用用户占用同一频率进行通讯
![[Pasted image 20250327165317.png]]


### 码分复用技术

码分复用技术用户挑选不同的码型，从而保证了各用户之间不会互相干扰，可以同时和同频率通信，现代移动设备和基站都采用类似技术

#### 工作原理
![[Pasted image 20250327170456.png]]

S站有单独的码片序列作为该站发送的唯一标识，将每个比特时间划分为m个段称为码片，如果一个站要发送比特1则发送自己的码片如果发送0则将码片取反，T站也相同，此时他们共用同一个信道进行通讯，叠加后的讯号为$S_x+T_x$
接收端对这个总的信号高作图上的运算，如果接收端用码片$S$进行运算那么得到的规格化内积$S\cdot S_x$就是需要接收的信号，而$S\cdot T_x$则全显示为0
