言语理解的解题原则是注重选项对于材料概括的完整性和侧重有没有符合材料本身所想表达的



- **找重点**，找到文章所要表达的关键句 
   - **转折之后是重点**，例如虽然...但是...，然而....，转折之后才是表达的重点内容
![[Pasted image 20251104143538.png]]


- **因果关系**，材料当中有明显的因果关系，**结论是重点**

![[Pasted image 20251104143856.png]]

如果选项当中同时有对结论和原因的概括优先选择这项，下图所示选更合理，因为A选项同时阐述了原因和结论

![[Pasted image 20251104143949.png]]


- **找对策**，材料如果是关于解决某个问题而提出解决方案的话，选项应该选择概括解决方案的选项，也就是**怎么做**

![[Pasted image 20251104144510.png]]


- **并列关系**，选项当中含有并列关系，要注意选项是否对于所有提到的并列点都有概括到，应当排除片面性的选项

![[Pasted image 20251104144739.png]]

![[Pasted image 20251104144815.png]]

**可能出现的特征**

![[Pasted image 20251104144915.png]]




- **辅助论证**，材料当中可能援引其他例子，或者采用一些论证方法来论证主要观点，做题者有可能会将为了论证而采用的论据当作主要论点，需要特别注意

![[Pasted image 20251104145410.png]]

例如下图这道题，材料大部分都为举例论证，‘总之’后面才是主要结论
